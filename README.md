 AI Language Translation Tool

The AI Language Translation Tool is a Python-based application developed for translating text from one language to another using Google Translator API. The project provides a simple and user-friendly graphical interface where users can enter text, choose source and target languages, and instantly get translated results.

This project is developed using Python, Tkinter, and the `deep-translator` library. It supports multiple languages such as English, Urdu, French, Hindi, German, and Spanish. The application also includes advanced features like a copy button for copying translated text easily.

The main objective of this project is to demonstrate the practical implementation of Artificial Intelligence and Natural Language Processing (NLP) in real-world language translation systems.

 Features
* Translate text into multiple languages
* User-friendly GUI interface
* Fast and accurate translation
* Copy translated text feature
* Supports multiple language selections

 Technologies Used

* Python
* Tkinter
* deep-translator
* Google Translator API

 Objective

The objective of this project is to create an intelligent language translation system that helps users easily understand and translate different languages using AI-based translation techniques.
